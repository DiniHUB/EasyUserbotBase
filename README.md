# EasyUserbotBase
Easy-to-use userbot framework for beginners based on Telethon.

# Features
* **Auto Dependencies Installation**
* **Auto Modules Loader**
* **Auto Config Setup**
* **Custom Events**
* **Colored Logs**
* **Elegant Code**

# How To Use

**Windows**: ``` py -m userbot ```  
**Linux**: ``` python3 -m userbot ```
